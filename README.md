# Amazon Clone Landing Page

A simple static Amazon-inspired storefront built with HTML and CSS. This project focuses on layout, visual structure, and product-card styling rather than backend functionality.

## Overview

The page includes:

- A dark top navigation bar with logo, delivery location, search, language, account, returns, and cart sections
- A secondary menu bar with category links
- A hero banner with a promotional message
- Multiple product showcase sections laid out as cards
- A deals section with featured electronics and gadgets
- A footer with informational links and copyright text

## Files

- `pro-2.html` - Main HTML structure for the page
- `pro-2.css` - Styling for the layout, colors, spacing, and components
- Image assets - Local product and branding images used throughout the page

## Features

- Fully static front-end layout
- Amazon-style UI inspired by the Amazon homepage
- Responsive-like structure built with flexbox-based sections
- Local images for banners, product cards, and logo
- Font Awesome icons loaded from a CDN

## How to Run

1. Open `pro-2.html` in any modern web browser.
2. Make sure the image files and `pro-2.css` stay in the same folder as the HTML file.
3. If you want to edit the design, update the HTML structure and CSS file together.

## Requirements

- A web browser such as Chrome, Edge, Firefox, or Safari
- No build tools or package installation are required

## Notes

- This project is for learning and practice only.
- Some custom HTML tags are used in the page structure, but the page still renders as a static front-end mockup.
- Several links are placeholders and can be replaced with real destinations if needed.
